# Tic Tac Toe

My first Ruby project, built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## How to run

Make sure you have Ruby installed, then clone the repo:

```bash
git clone git@github.com:shizzic/tic_tac_toe.git
cd tic_tac_toe
bundle install
ruby tic_tac_toe.rb --without development
```

Two players take turns entering a position (1-9) to place their mark on the board. First to get three in a row wins!
